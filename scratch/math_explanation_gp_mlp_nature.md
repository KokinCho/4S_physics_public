# GPR と MLP の予測曲線が数学的に反映しているもの

本ファイルでは、ガウス過程回帰 (GPR) と多層パーセプトロン (MLP) の予測曲線が、それぞれ数理的に何を反映して出力されているのか、および授業スライド（講義資料）のどの部分を参照すれば良いかを詳しく解説します。

---

## 1. ガウス過程回帰 (GPR) の予測曲線が表すもの

### 数学的本質：確率過程の「事後平均（条件付き期待値）」のカーネル補間
GPR の予測曲線は、無限次元の関数に対する確率分布（ガウス過程）から得られる**「観測データを与えられた下での事後予測平均」**をプロットしたものです。

新しいテスト入力 $x_*$ に対する予測平均値 $m_*(x_*)$ は、以下の公式で計算されます：
$$m_*(x_*) = \bm{k}_*(x_*)^T \left( K(X, X) + \sigma_n^2 I_n \right)^{-1} \bm{y}$$

ここで：
- $\bm{y}$ は学習データのターゲットベクトル
- $K(X, X)$ は学習データ間の共分散行列（グラム行列）
- $\bm{k}_*(x_*) = \left( k(x_1, x_*), \dots, k(x_n, x_*) \right)^T$ は学習データとテスト点 $x_*$ 間の共分散ベクトル

**何を反映しているか：**
1. **データ点との類似度に基づく重み付き線形結合**：
   上記の予測平均の式は、係数ベクトルを $\bm{\alpha} = (K + \sigma_n^2 I_n)^{-1} \bm{y}$ と置くと、以下のように書き直せます。
   $$m_*(x_*) = \sum_{i=1}^n \alpha_i k(x_i, x_*)$$
   すなわち、予測曲線は「各学習データ点 $x_i$ とテスト点 $x_*$ の共分散（カーネル関数）の重み付き線形結合」そのものです。
2. **事前仮定（カーネル）による平滑化**：
   RBFカーネルを用いることで、「近い点同士は似た値を取る（滑らかに変化する）」という事前情報が暗黙的に反映され、データ点が少ない $n=10$ の段階でも滑らかなサインカーブに近い予測が得られます。

### 授業スライドの参照先：『Intro_ML5.pdf』
- **p. 5 「ガウス過程」**：
  任意の入力点の組に対する関数値が多次元ガウス分布に従う確率変数として関数をモデリングする基本概念。
- **p. 14 「ガウス過程回帰」**：
  結合分布 $p(\bm{t}_{N+1})$ から条件付き分布（事後分布）を求める定式化。
- **p. 18 「ガウス過程に関する予測値の公式」**：
  事後平均 $m_{N+1}$ と事後分散 $\sigma_{N+1}^2$ の導出式。多次元正規分布の条件付き分布の公式（Schur 相補空間）との対応関係。
- **p. 20 「ガウス過程回帰の例」**：
  データ点が得られた際に、予測平均と信頼区間がどのように変化するかのビジュアル的な例。

## 2. 多層パーセプトロン (MLP) の予測曲線が表すもの

### 数学的本質：非線形な「決定論的写像」のパラメータ最適化（点推定）
MLP の予測曲線は、多数のパラメータ（重み $\bm{w}$ とバイアス $\bm{b}$）から構築された非線形な関数モデルによる**「点推定としての予測写像」**をプロットしたものです。

入力 $x_*$ に対する予測値 $\hat{y}(x_*)$ は、活性化関数（今回は `tanh`）を介した順伝播計算によって求まります：
$$\hat{y}(x_*) = \sum_{j=1}^M w_{j}^{(2)} \tanh \left( w_{j}^{(1)} x_* + b_{j}^{(1)} \right) + b^{(2)}$$

このパラメータ群 $\bm{\theta} = \{\bm{w}, \bm{b}\}$ は、学習データ上の誤差関数（今回は L2 正則化を含む二乗誤差）を最小化するように最適化（点推定）されます：
$$E(\bm{\theta}) = \frac{1}{2} \sum_{i=1}^n \left( f_{\text{MLP}}(x_i; \bm{\theta}) - y_i \right)^2 + \frac{\alpha}{2} \lVert \bm{\theta} \rVert_2^2$$

**何を反映しているか：**
1. **正則化付き最小二乗誤差の局所解（ローカルミニマ）**：
   MLPは確率的なアプローチ（事後分布の計算）ではなく、決定論的な最適化を行います。特にデータ数が少ない $n=10$ の場合、自由度（2701個のパラメータ）がデータ数に対して過剰な「劣決定系」となるため、誤差を最小化するパラメータの組み合わせが無限に存在し、最適化ソルバー（L-BFGS）が初期値や乱数の影響でたどり着いた「ある1つの局所解」の形をそのまま反映してうねってしまいます。
2. **モデル表現力の高さと制約の無さ**：
   GPRのように「RBFカーネルで相関が減衰する」といった強い平滑化の制約が直接コード化されていないため、パラメータが過剰適合（過学習）し、データの空白域で激しい振動が生じやすくなります。

### 授業スライドの参照先：『Intro_ML7.pdf』
- **p. 7 「2層ニューラルネットワーク」**：
  入力、中間層、出力層、および活性化関数（$\sigma$ や $h$）を用いたモデルの表現式。
- **p. 11 「多層ニューラルネットの万能性」**：
  中間層の素子数を増やす、または層数を増やすことで、任意の連続関数を近似できる表現能力についての言及。
- **p. 13 「勾配法による学習」**：
  ガウス過程やSVMが「第1層の変換（カーネル）を固定することで最適化を凸（大域解が一意）にする」のに対し、ニューラルネットワークは「多層のすべてのパラメータを最適化するため、大域的最適解を得ることが一般に難しく、局所解に依存する」という記述。
- **p. 16 「誤差逆伝播法」**：
  最小化する対象である二乗誤差関数 $E(\bm{w})$ の定義式。
- **p. 23 「簡単な例」**：
  1次元入力における `tanh` を用いた2層ニューラルネットの具体的な関数式（今回のモデルと同一構造）。

## 3. まとめ：両者のアプローチの違い

| 比較項目 | ガウス過程回帰 (GPR) | 多層パーセプトロン (MLP) |
| :--- | :--- | :--- |
| **モデル分類** | 非パラメータモデル（確率的・ベイズ的） | パラメータモデル（決定論的・最適化ベース） |
| **予測曲線** | 既知のデータとの類似度（カーネル）に基づく**事後平均** | 訓練データで誤差最小化された**パラメータによる関数写像** |
| **データ不足時** | カーネルによる滑らかさの事前制約により、**真の形状に沿った安定した補間**となる | 自由度過剰による劣決定系となり、**局所解に依存した不安定なうねり（過学習）**を示す |
| **スライド** | 『Intro_ML5.pdf』 (p. 5, 14, 18) | 『Intro_ML7.pdf』 (p. 7, 13, 16) |

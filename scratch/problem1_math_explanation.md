# L1/L2 正則化の数学的背景と特徴量選択のメカニズム

この文書では、レポート問題1（正則化の比較）の背景にある数学的なメカニズムを詳細に解説します。Markdown Preview を開くことで数式が綺麗にレンダリングされます。

---

## 1. 定式化

線形回帰における計画行列を $X \in \mathbb{R}^{M \times N}$、目的変数を $y \in \mathbb{R}^M$、パラメータを $w \in \mathbb{R}^N$ とします。

### 1.1 正則化なし（最小二乗法: OLS）
通常の最小二乗法では、二乗誤差（残差平方回）を最小化します：
$$ E_{\mathrm{OLS}}(w) = \frac{1}{2M} \|y - Xw\|_2^2 $$

### 1.2 L1 正則化（Lasso 回帰）
L1ノルム（パラメータの絶対値の和）をペナルティとして加えます：
$$ E_{\mathrm{Lasso}}(w) = \frac{1}{2M} \|y - Xw\|_2^2 + \alpha \sum_{j=1}^N |w_j| $$
ここで、$\alpha > 0$ は正則化の強度を制御するハイパーパラメータです。

### 1.3 L2 正則化（Ridge 回帰）
L2ノルム（パラメータの二乗和）をペナルティとして加えます：
$$ E_{\mathrm{Ridge}}(w) = \frac{1}{2M} \|y - Xw\|_2^2 + \frac{\alpha}{2} \sum_{j=1}^N w_j^2 $$

---

## 2. なぜ L1 正則化（Lasso）では重みが完全に 0 になるのか？

Lasso 回帰の最大の特徴は、いくつかのパラメータが**完全に 0 になる（スパースな解が得られる）**ことです。これには幾何学的な解釈と、代数的な解釈の2つのアプローチがあります。

### 2.1 幾何学的解釈（制約条件付き最適化）
正則化項を制約条件として書き換える（ラグランジュの未定乗数法に対応）と、最適化問題は以下のように書き換えられます。
- **Lasso (L1):** 
  $$ \min_{w} \frac{1}{2M} \|y - Xw\|_2^2 \quad \text{subject to} \quad \sum_{j=1}^N |w_j| \le C $$
- **Ridge (L2):** 
  $$ \min_{w} \frac{1}{2M} \|y - Xw\|_2^2 \quad \text{subject to} \quad \sum_{j=1}^N w_j^2 \le C $$

2次元平面 ($w_1, w_2$) で制約範囲を図示すると：
- **L1 の許容領域**は、$\pm$の軸上で角（頂点）を持つ**ダイヤモンド型（正方形）**になります。
- **L2 の許容領域**は、原点を中心とする**円**になります。

二乗誤差の等高線（楕円）が、この許容領域と初めて接する点が最適解となります。
- **L1（ダイヤモンド型）の場合:** 境界が「尖っている」ため、楕円がダイヤモンドの「角（軸上）」で接する確率が極めて高くなります。軸上で接するということは、どちらか一方の座標（例えば $w_1$）が完全に 0 になることを意味します。
- **L2（円）の場合:** 境界が滑らかなため、楕円がちょうど軸上で接する必然性はなく、一般には両方のパラメータが非ゼロのまま小さな値に収縮（Shrink）します。

---

### 2.2 代数的解釈（1変数における軟しきい値作用素の導出）
特徴量同士が互いに直交している（$X^T X = I$）と仮定すると、各パラメータ $w_j$ の最適化問題は独立に解くことができます。
OLSにおける解を $w_j^{\mathrm{OLS}}$ とします。

#### Lasso の解（Soft-Thresholding）
Lasso の目的関数を $w_j$ について微分します。絶対値関数 $|w_j|$ は $w_j = 0$ で微分不可能であるため、劣微分（subdifferential）を考えます。
最適化条件（$0 \in \partial E(w_j)$）を解くと、以下の **軟しきい値作用素（Soft-thresholding operator）** が得られます：
$$ w_j^{\mathrm{Lasso}} = S_{\alpha}(w_j^{\mathrm{OLS}}) = \text{sign}(w_j^{\mathrm{OLS}}) \max\left( 0, |w_j^{\mathrm{OLS}}| - \alpha \right) $$

グラフに描くと、この関数は $[-\alpha, \alpha]$ の範囲で**完全に 0** に張り付きます。
つまり、最小二乗解の絶対値が正則化強度 $\alpha$ よりも小さい場合、その重みは強制的に 0 にリセットされます。

#### Ridge の解
一方、Ridge の場合は目的関数が常に微分可能であり、容易に以下のように求まります：
$$ w_j^{\mathrm{Ridge}} = \frac{1}{1 + \alpha} w_j^{\mathrm{OLS}} $$
これは元のOLS解を定数倍（$\frac{1}{1+\alpha} < 1$）で縮小させるだけであり、$\alpha < \infty$ である限り、解が完全に 0 になることはありません。

## 3. その他の正則化：Elastic Net（問題 1.5 で提案）

L1 と L2 正則化のメリットを組み合わせたものが **Elastic Net** です。

### 3.1 定式化
$$ E_{\mathrm{ElasticNet}}(w) = \frac{1}{2M} \|y - Xw\|_2^2 + \alpha \left( \rho \sum_{j=1}^N |w_j| + \frac{1-\rho}{2} \sum_{j=1}^N w_j^2 \right) $$
ここで、$\rho \in [0, 1]$ は L1 と L2 の混合比率を定めるパラメータです。

### 3.2 Elastic Net が解決する Lasso の弱点
Lasso には以下の実用上の欠点があります：
1. **特徴量グループ選択（Grouping Effect）の欠ろ:**
   相関の非常に高い特徴量のグループ（例：身長、体重、BMIなど）が存在する場合、Lasso はその中からランダムに1つだけを選択し、残りの重みを 0 にしてしまいます。これによりモデルの解釈が不安定になります。
   Elastic Net は L2 ペナルティのブレンド効果（円型の制約）により、相関の強い特徴量グループをまとめて選択（グループ全体の重みを揃える）することができます。
2. **特徴量数の制約（$N > M$ 問題）:**
   サンプル数 $M$ よりも特徴量数 $N$ が多い場合、Lasso は最大でも $M$ 個の特徴量しか選択できません。Elastic Net は L2 ペナルティのおかげで、任意の数の特徴量を選択することが可能です。

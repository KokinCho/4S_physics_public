# 二重降下現象（Double Descent）のネーミングの理由と「1回目の降下」

「二重降下現象（Double Descent）」という名前は、モデルの複雑さ（今回の線形回帰ではパラメータ数 $p$）を増やしたときに、テスト誤差（Test MSE）が**「下落するフェーズが2回現れる」**ことから名付けられました。

古典的な機械学習の理論と、現代の過学習・過パラメータ（Overparameterized）理論を繋ぐ非常に興味深い現象です。それぞれの降下フェーズがグラフのどの部分に対応するかを詳しく解説します。

---

### 二重降下グラフの構成要素

モデルのパラメータ数 $p$（または $\gamma = p/n$）を $0$ から増やしていったときのテスト誤差の挙動は、大きく分けて以下の**3つのフェーズ**（降下 $\rightarrow$ 上昇 $\rightarrow$ 再降下）に分かれます。

```
テスト誤差
  ^
  |      (古典的領域: U字型)       |  (現代的・過パラメータ領域)
  |                                |
  |   \                            |       /|
  |    \                 /|        |      / |
  |     \               / |        |     /  |
  |      \             /  |        |    /   |
  |       \___________/   |        |   /    \
  |                       |        |  /      \_________
  |                       |        | /
  +-----------------------+--------+-------------------> パラメータ数 p
   <---【1回目の降下】--->  <-上昇->  <--【2回目の降下】-->
                         (p=nで発散)
```

#### 1. 1回目の降下（古典的な学習フェーズ）
* **範囲**: $p$ が非常に小さい状態から、古典的な最適モデルの複雑さに至るまで（$\gamma = p/n$ が $0$ から U字の底まで）。
* **説明**: パラメータ数 $p$ を増やすことで、モデルの表現力（バイアスの削減）が向上し、テスト誤差が**減少（降下）**します。これが**「1回目の降下」**です。

#### 2. 上昇（過剰適合・オーバーフィッティングフェーズ）
* **範囲**: U字の底から、データ数とパラメータ数が等しくなる $p = n$ ($\gamma = 1$) に至るまで。
* **説明**: パラメータ数をさらに増やすと、モデルが必要以上にノイズまで学習してしまい、バリアンス（ばらつき）が急増します。テスト誤差は上昇に転じ、パラメータ数がデータ数と一致する $p = n$ の境界（**補間境界: Interpolation Boundary**）で、分母がゼロに近づくため誤差が極大（あるいは無限大に発散）となります。
* *※ここまでは、古典的な統計学でよく知られている「バイアス・バリアンスのトレードオフ（U字型の曲線）」そのものです。*

#### 3. 2回目の降下（現代的な過パラメータフェーズ）
* **範囲**: $p > n$ ($\gamma > 1$) の領域。
* **説明**: パラメータ数がデータ数を超える（モデルがデータを完全に丸暗記できる表現力を持つ）と、方程式の解が無数に存在するため、推定時に「パラメータの L2 ノルムを最小化する」という暗黙的な正則化（最小ノルム解）が働きます。
  パラメータ数 $p$ がさらに増えれば増えるほど、この「L2 ノルム最小化」がより効果的に機能し、パラメータ空間の自由度が高まることで、バリアンス（過剰適合による跳ね返り）が抑え込まれます。その結果、テスト誤差は再び**減少（降下）**し始めます。これが**「2回目の降下」**です。

---

### まとめ

* **1回目の降下**: モデルの表現力が足りない状態（Underfitting）から、適切な表現力を得ることでバイアスが減少し、誤差が下がるプロセス。
* **2回目の降下**: データ数を超えるパラメータを持つ超巨大なモデルにおいて、L2 ノルム最小化という暗黙の正則化が効いてバリアンスが減少し、再び誤差が下がるプロセス。

このように、「下がって、上がって、また下がる」という波打つような曲線を描くため、**「二重降下現象（Double Descent）」**と呼ばれています。
